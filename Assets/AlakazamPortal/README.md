# Alakazam Portal

Unity SDK for real-time AI stylization. Transform any greybox scene into stylized game renders.

## Installation

Add via Package Manager → Add package from git URL:
```
https://github.com/yourorg/alakazam-portal.git?path=Assets/AlakazamPortal
```

## Quick Start

1. **AlakazamPortal → Add to Current Scene** from the menu
2. Select `AlakazamController` in Hierarchy
3. Enter your **Server URL**
4. Press Play, then **Space** to start streaming

## Requirements

- Unity 6000.0+
- Active Alakazam key
