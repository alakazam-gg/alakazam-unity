# Alakazam Portal

Unity SDK for real-time AI stylization. Transform any greybox scene into stylized game renders.

## Installation

Add via Package Manager → Add package from git URL:
```
https://github.com/yourorg/alakazam-portal.git?path=Assets/AlakazamPortal
```

## Quick Start

1. **AlakazamPortal → Add to Current Scene** from the menu
2. Select `AlakazamController` in Hierarchy
3. Enter your **Server URL**
4. Press Play, then **Space** to start streaming

## Requirements

- Unity 6000.0+
- Active Alakazam key

## Recommended Demo Assets

To explore and experiment with the Portal, we recommend (among others) these free low-poly asset packs from the Unity Asset Store:

| Asset Pack | Description |
|------------|-------------|
| [SimplePoly City - Low Poly Assets](https://assetstore.unity.com/packages/3d/environments/simplepoly-city-low-poly-assets-58899) | Urban environments, buildings, vehicles |
| [LowPoly Environment Pack](https://assetstore.unity.com/packages/3d/environments/lowpoly-environment-pack-99479) | Nature scenes, trees, rocks |
| [Elementary Dungeon Pack Lite](https://assetstore.unity.com/packages/3d/environments/dungeons/lite-dungeon-pack-low-poly-3d-art-by-gridness-242692) | Dungeon interiors, props |

Low-poly assets work great with Alakazam's AI stylization, as the clean geometry transforms beautifully into any art style.

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| Space | Start/Stop streaming |
| 1-8 | Select style preset |
| B | Split-screen comparison |
| R | Capture screenshot |
| Tab | Toggle UI |
